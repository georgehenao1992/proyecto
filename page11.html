MEDIDAS DE TENDENCIA CENTRAL

Son parámetros estadísticos que indican valores cuyo objetivo es resumir la información para un conjunto de datos, es decir, son representantes de un conjunto de datos. Las medias de tendencia central mas conocidas son:

Media Aritmética, la Mediana y la Moda.

Medidas de Tendencia Central para Datos NO Agrupados

MEDIA ARITMETICA

Valor numérico que corresponde al cociente de la suma de los datos y el numero total de observaciones (promedio) se denota con x = .......  

no olvidar donde n es el numero de le elementos de la muestra.

Ejemplo:

Si se considera el numero de hijos de 7 familias con los siguientes resultados:
1, 2, 2, 4, 5, 5, 6, la media aritmética del numero de hijos es: 3,6

lastimosamente el resultado no corresponde a un numero entero positivo  (numero de hijos) en este caso el valor de la media aritmética se puede aproximar. 

MEDIANA

Se define como el valor central de un conjunto de datos ordenados de manera creciente o decreciente. en el caso que el numero sea par, la mediana corresponde a la media aritmética de los dos valores central, se denota como Me.

Ejemplo:

las notas de 6 alumnos de un curso de ciencias son las siguientes:
2,0 - 3,0 - 3,0 - 4,0 - 4,0 - 5,0.

la mediana es 3,5

MODA

La moda de un conjunto de observaciones corresponde a aquel datos que tiene la mayor frecuencia. se denota como Mo.
Del ejemplo anterior la moda es 3,0 y 4,0
